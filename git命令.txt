本地与远程仓库的绑定
打开终端并导航到您的本地 Git 仓库所在的目录。

然后，使用以下命令将本地仓库与 GitHub 仓库进行关联：

1、$ git remote add origin https://github.com/username/repository.git
其中 username 是您的 GitHub 用户名，repository 是您要关联的仓库的名称。


2、如果您使用的是 SSH 协议，则使用以下命令：$ git remote add origin git@github.com:username/repository.git


接下来，使用以下命令将本地更改推送到 GitHub 仓库：
3、$ git push -u origin master


clone：
$ git clone https://github.com/username/repository.git
$ git clone git@github.com:username/repository.git
